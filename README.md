# Student Data Management System

## Description
The Student Data Management System is a simple desktop application developed using Python and Tkinter. This project is designed for learning purposes and demonstrates how to manage student records through a graphical user interface (GUI).

---

## Features
- Add new student records
- View student details
- Update student information
- Delete student records
- Simple and user-friendly interface

---

## Technologies Used
- Python
- Tkinter (GUI library)

---

## How It Works
This application allows users to manage student data through a graphical interface instead of using command line input. Users can enter student details such as name, roll number, and other information, which can then be stored, modified, or removed.
